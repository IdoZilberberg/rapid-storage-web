## Setup
* Install [Node/npm](https://nodejs.org/en/) 
* Install Angular: `npm install -g @angular/cli@latest` (may need `sudo`)
* Clone repo: `git clone https://github.com/IdoZilberberg/rapid-storage-web.git`
* Run `npm install`
* Run `ng serve`
